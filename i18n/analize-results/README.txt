This folder contains the files which are created analyzing the openpetra-template.

The following files are generated:
analysissummary - contains a summary of the different item-types

firstoccurence - the first ocurrence of an item
firstoccurence_xword - the first ocurrence of all items consisting of only x word(s)

doubles - all items with their derivates (same item but with different additional characters.,:;&() or casing)
