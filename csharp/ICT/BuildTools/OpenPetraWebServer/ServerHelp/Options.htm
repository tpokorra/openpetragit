Full GUI Startup Options

This topic applies only to the Full GUI

There are three options that you can configure that change the behaviour of the application.

  • Start Automatically: If you check this item, the server will start as soon as the application loads.
  • Hide Window at Startup: If you check this item, the application window will be displayed for five seconds and will then be automatically minimised to the system tray. You can display the window again by accessing the system tray icon menu.
  • Allow Remote Connections: By default the web server only accepts requests for pages from the local computer. Check this option if you want to make your web server public. It is not advisable to make this server accessible to the internet.

Note that some options can only be changed when the server is stopped.
