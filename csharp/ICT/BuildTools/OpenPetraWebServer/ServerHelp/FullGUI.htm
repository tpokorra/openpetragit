The Main Window of the Full GUI

This topic applies only to the Full GUI

The main window displays the list of sites that have been configured. There is a link that you can click (if the server is started) that will take you to the home page of each site.

If the main window is not visible, you restore it by one of the following:

  • double-clicking the web server icon in the system tray
  • right-clicking on the System Tray icon and choosing the 'Open' or 'Bring to Front' menu items

The main window features:

  • a menu bar at the top.
  • a button bar beneath the menu bar.
  • a status bar at the bottom.
  • an icon in the System Tray with a context-sensitive pop-up menu

Some of the most important actions on this window are to

  • Start or Stop the server.
  • Add or remove sites.
  • Edit the properties of one of the existing sites.
  • Change options such as allowing remote connections or minimising the window at startup.

All the settings and information that you enter on the main window is automatically preserved when you exit the application.
