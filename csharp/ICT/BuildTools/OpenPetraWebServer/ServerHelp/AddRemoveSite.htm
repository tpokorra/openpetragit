Adding or Removing Web Sites

This topic applies only to the Full GUI

You can run any number of web sites from the web server. The only restriction is that they must all use a different port number. Add a new web site to the web server by clicking on the main window menu and selecting the Add Site ... item. The Web Site Properties dialog will be displayed. You need to enter the following information.

  • A Friendly Name: this is a unique name that identifies this entry in the list of installed web sites. The name does not form part of the web address.
  • The Physical Path: this is the actual location on your hard drive that will be the root of the site.
  • The Virtual Path: this is the name of the site that will appear after localhost in the URL. You can leave this entry blank.
  • The Port Number: this is the HTTP port that the server will listen on for web requests. The standard port is 80. Other favourite choices are 8080, 8888, 8088 but you can use any number, usually above 50000. If you have more than one web site you must use a different port number for each one.
  • The Default Page: You can leave this blank, but if you do the server will try to locate a 'default.htm' or 'default.aspx' page or similar. If it cannot find one of its default documents it will display a directory listing from which you can browse to a listed page. You will usually want to supply a default page if the site home page is not one of the standard documents.

Note that you can only add a new web site when the server is stopped.

You can remove a web site by selecting the site in the main window list and clicking the Remove item from the Seleced Site menu.

The application saves your web site and option settings to disk. When you run the application again it will start again from where you left off.
