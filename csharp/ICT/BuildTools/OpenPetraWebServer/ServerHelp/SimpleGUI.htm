The Simplified GUI

This topic applies only to the Simplified GUI

When you launch the Open Petra Web Server by adding command line parameters the application displays a simplified GUI because you can only create a single web site. If you need to host more than one web site then you must launch the application multiple times - once for each site.

The GUI shows the properties of the web site that you have started. The GUI automatically starts the server and then is minimised to an icon in the System Tray.
