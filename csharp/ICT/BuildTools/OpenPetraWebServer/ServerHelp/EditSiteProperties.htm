Editing the Properties of a Web Site

The Simplified GUI does not permit you to edit the properties of the web site that was specified using the command line.

The Full GUI does provide the ability to edit site properties.

You change the properties of a particular site either

  • By clicking the site in the main window list and selecting the Properties menu from the Selected Site menu item.
  • By clicking the site in the main window list and then clicking the Properties button on the button bar.
  • By double-clicking the site in the site list on the main window.
  • By right-clicking on the icon in the System Tray to bring up the context menu, then selecting the Properties item and then the site whose properties you want to change.

You can only modify the properties of a site when the server is stopped, but you can view the properties at all times.
