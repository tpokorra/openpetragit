The System Tray Icon

You will usually run the web server so that its main window is hidden but you can always access the application by clicking its icon in the system tray at the bottom right of the screen.

Double clicking with the left mouse button will bring up the main window, as will right-clicking and selecting Open or Bring To Front. The menu that is displayed with the right-click action will perform all of the most useful program features.

  • Close will minimise the application to the System Tray
  • Start or Stop starts or stops the web server
  • Browse will display the web site in your default browser.
  • Help Contents shows this help
  • Help About shows version information about the application
  • Properties (on the Full GUI only) shows the properties for the specified web site
  • Exit will quit the application. This is the only way to close the application. Clicking the close button on the main GUI window simply minimises the application back to the System Tray.
