Starting and Stopping

When the site parameters are entered on the command line

When you run the application from the command line the server starts as soon as the application starts. You stop the server (and restart it again) using the System Tray right-click menu, although it will be rare that you need to start or stop the server at all.

When the application is run with the Full GUI

The web server is not automatically running simply because the web server application is running. It is necessary to explicitly start the server by selecting the Start Server menu option, either from the main menu or from the system tray right mouse menu or by clicking the blue start button on the button bar.

When the server is running there are several actions that are disabled. For example, once the server is running it is not possible to modify a web site's connection properties. In order to do so you must first select the Stop Server menu item or click the red stop button. If you hover the mouse over the web server icon in the system tray, the tool tip will tell you if the server is running or not.

You can choose to have the server start automatically as soon as the web server application starts by making the appropriate choice on the Options menu.
