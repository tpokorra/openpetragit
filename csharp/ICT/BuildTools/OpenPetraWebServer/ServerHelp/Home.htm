The Open Petra Web Server

The Open Petra web server application allows a client to communicate with a web site or a web service. When the web server is running you can view web sites installed on this computer. Optionally you can allow other computers to see the web sites on this computer.

You will usually run the web server in the background but a choice of two user interfaces is available to manage the sites.

  • The Full GUI - the full GUI supports multiple web sites
  • The Simplified GUI - the simplified GUI is used when you run the application from the command line

In normal use the GUI is not visible and the application simply shows an icon in the system tray. This icon has a pop-up menu that provides access to all the application functionality.

You browse to a web site on your computer by using a web address such as http://localhost:57912/home.htm (this link will actually take you to the home page for this application's help). The current version of Open Petra Web Server only supports http connections. Possibly in the future it may be extended to include https (SSL) connections. This is the main reason that it should not be used anywhere where a secure connection is necessary.
