Command Line Parameters

If you launch Open Petra Web Server with no command line parameters, you will be presented with the Full GUI. You cannot have more than one instance of the Full GUI open at one time. If you try to do so, you will simply activate the existing instance. The Full GUI enables you to set up and manage multiple web sites in one place.

However sometimes it will suit your workflow better to launch multiple single instances of the web server. In this case you add the server details as command line parameters and each instance has its own simplified GUI. The command line format is as follows. Note that there is always at least one space between a switch and its value. See, for example the Port Number below.

These parameters are always required

  • The Physical Path: express this as a fully qualified path, in quote marks if necessary
  • The Port Number: express this as, for example,  /p  8080

These parameters are optional

  • The Virtual Path: express this as, for example,  /v  "/MyMainSite" - note that this must always be in quotation marks
  • The Default Page: express this as, for example,  /d  MyService.asmx
  • Accept Remote Connection: express this as  /r
  • Quiet startup: express this as  /q - this suppresses any message that the server is already running and the port is in use

You can specify the parameters in any order. If you enter a bad command line, the Open Petra Web Server will display a window that reminds you of the syntax. Note that this will also be displayed if you enter a physical path that does not exist.

When the server application starts it checks that the specified port is not in use. If the port is not available, it assumes that an instance of the server is already running, displays a message (unless /q was used) and the new instance closes immediately, leaving the existing instance running. This means that you can issue the command to run the application as many times as you like. You can never get more than one instance of the application trying to use the same port at the same time.
